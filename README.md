# Vue PC 端云音乐项目

### 1. 项目介绍
LiliYao-NeteaseMusic（未完成版）

如果对各位朋友有用的话, 希望可以给个star⭐

#### 技术栈
- 使用 Vue 框架进行开发
- 使用 axios 进行网络请求
- CSS 使用 FLEX 进行布局
- 项目组件库使用 ElementUI

#### 项目使用库
- ElementUI
- Vuex

#### 项目目标
- 使用 Vue 框架开发 PC 端云音乐项目
